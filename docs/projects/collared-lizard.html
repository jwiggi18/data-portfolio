Mapping Crotaphytus collaris Sightings in Oklahoma

Question

Where are iNaturalist sightings of Crotaphytus collaris concentrated in Oklahoma, and what patterns or biases appear in the data?

Why this project matters

Species occurrence data are often used to infer distribution, but observational datasets can reflect both biology and human sampling behavior. This project explores both.

Data source

iNaturalist observations API. The dataset was pulled programmatically from the observations endpoint and flattened into a CSV for analysis. Link to code here.

Approach

  • Filter observations to Oklahoma
  • Clean location data
  • Plot points on a map
  • Examine clustering and possible sources of bias

Basic cleaning

df = df.dropna(subset=["latitude", "longitude"]).copy()

df["year"] = df["observed_on"].dt.year

Map Styling function

def style_oklahoma_map(fig, title):
    fig.update_traces(
        marker=dict(
            size=7,
            opacity=0.75,
            line=dict(width=0.4, color="black")
        )
    )

    fig.update_layout(
        title=dict(
            text=title,
            x=0.5,
            xanchor="center",
            y=0.95
        ),
        paper_bgcolor="white",
        plot_bgcolor="white",
        margin=dict(l=10, r=10, t=40, b=10),
        legend=dict(
            title="Quality grade",
            x=0.0,
            y=0.5,
            xanchor="left",
            yanchor="middle",
            bgcolor="rgba(255,255,255,0.6)",
            bordercolor="black",
            borderwidth=0.5
        )
    )    
    
    fig.update_geos(
        scope="usa",
        center=dict(lat=35.5, lon=-97.5),
        projection_scale=6,
        bgcolor="white",
        showland=True,
        landcolor="white",
        showcountries=True,
        countrycolor="black",
        countrywidth=0.8,
        showsubunits=True,
        subunitcolor="black",
        subunitwidth=1.2,
        showlakes=False,
        showcoastlines=False
    )

    return fig

Key takeaways

  • Observations cluster in accessible or highly visited locations (Wichita Mountains Wildlife Refuge & The Osage Reservation)
  • Reported occurrence does not necessarily equal true distribution
  • Visualization helps reveal likely sampling bias quickly
